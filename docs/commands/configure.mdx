## Synopsis

```bash
plerion configure [subcommand]
```

## Interactive wizard (default)

```bash
plerion configure
```

Prompts for:
- **Profile name** (default: `default`)
- **API key**
- **Region** (`au`, `sg1`, `in1`, `us1`)
- **Default output format** (`table`, `json`, `yaml`, `text`)

Writes to `~/.plerion/credentials` and `~/.plerion/config`.

## Subcommands

### list

```bash
plerion configure list
```

Lists all configured profile names.

## Examples

```bash
# Set up the default profile
plerion configure

# Set up a named profile
plerion configure
# > Profile name: prod
# > API key: pk_live_...
# > Region: us1
# > Output format: json

# List all profiles
plerion configure list
```

### Example output

```
default
prod
staging
```
