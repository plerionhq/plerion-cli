## From GitHub releases (recommended)

Download the latest binary for your platform from [Releases](https://github.com/plerionhq/plerion-cli/releases):

| Platform | Binary |
|---|---|
| macOS (Apple Silicon) | `plerion-aarch64-apple-darwin` |
| macOS (Intel) | `plerion-x86_64-apple-darwin` |
| Linux x86_64 | `plerion-x86_64-unknown-linux-musl` |
| Linux ARM64 | `plerion-aarch64-unknown-linux-musl` |
| Windows x64 | `plerion-x86_64-pc-windows-msvc.exe` |

<Tabs>
  <Tab title="macOS (Apple Silicon)">
    ```bash
    curl -L -o plerion https://github.com/plerionhq/plerion-cli/releases/latest/download/plerion-aarch64-apple-darwin
    chmod +x plerion
    sudo mv plerion /usr/local/bin/
    ```
  </Tab>
  <Tab title="macOS (Intel)">
    ```bash
    curl -L -o plerion https://github.com/plerionhq/plerion-cli/releases/latest/download/plerion-x86_64-apple-darwin
    chmod +x plerion
    sudo mv plerion /usr/local/bin/
    ```
  </Tab>
  <Tab title="Linux x86_64">
    ```bash
    curl -L -o plerion https://github.com/plerionhq/plerion-cli/releases/latest/download/plerion-x86_64-unknown-linux-musl
    chmod +x plerion
    sudo mv plerion /usr/local/bin/
    ```
  </Tab>
  <Tab title="Linux ARM64">
    ```bash
    curl -L -o plerion https://github.com/plerionhq/plerion-cli/releases/latest/download/plerion-aarch64-unknown-linux-musl
    chmod +x plerion
    sudo mv plerion /usr/local/bin/
    ```
  </Tab>
  <Tab title="Windows">
    Download `plerion-x86_64-pc-windows-msvc.exe` from the [releases page](https://github.com/plerionhq/plerion-cli/releases) and add it to your `PATH`.
  </Tab>
</Tabs>

Verify the installation:

```bash
plerion --help
```

## From source

Requires Rust 1.70+ and Cargo:

```bash
git clone git@github.com:plerionhq/plerion-cli.git
cd plerion-cli
cargo build --release
sudo cp target/release/plerion /usr/local/bin/
```
